## ☀️ 太阳能供电技术 deep dive

### 1. "双核" 可能的含义

#### 假设 A: 双 CPU 架构 (最可能)

```
┌─────────────────────────────────────┐
│  应用处理器 MPU (Linux)             │
│  - 屏幕 UI                           │
│  - 语音识别 (ASR)                   │
│  - LLM 推理 (本地/云端)              │
│  - 网络连接                         │
├─────────────────────────────────────┤
│  微控制器 MCU (RTOS)                │
│  - 电源管理 (太阳能充电 + 电池)     │
│  - GPIO 控制 (电机, LED)            │
│  - 低功耗待机 (太阳能优化)           │
│  - 实时响应 (按键, 传感器)           │
└─────────────────────────────────────┘
```

**优势**:
- MCU 待机功耗 < 1mA，太阳能板阴天也能维持
- MPU 按需唤醒 (语音/触摸触发)
- 符合"双核"营销话术

**参考设计**: 类似 Raspberry Pi Zero + Arduino 搭配

#### 假设 B: 双太阳能电池板

- **主光伏板**: 20-30W，车顶大板，晴天快速充电
- **辅光伏板**: 5-10W，机器人本体小板，阴天/室内维持
- 双路 MPPT 独立充电管理

#### 假设 C: 双电池系统

- **主电池**: 18650 磷酸铁锂 (10Ah, 3.2V) - 用于电机、屏幕
- **辅电池**: 超级电容或锂电 (2Ah) - 用于快速启动、平抑功率

---

### 2. 典型太阳能供电系统设计

**场景设定**: 车载机器人，室内部使用，日均功耗 ~10-15Wh

```
┌─────────────┐
│ 太阳能电池板 │ 20-40W (18-22V 开路电压)
│  (车顶/机身)│
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ MPPT 控制器 │ 最大功率点跟踪，效率 > 95%
│ (20-40A)    │ 支持锂电/磷酸铁锂
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  电池组     │ 3.2V 磷酸铁锂 10Ah (32Wh)
│ (18650×4)   │ 或 18650×3 (7.5Ah)
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ DC-DC 降压  │ 5V/3A (15W) 给 MPU
│  降压模块   │ 12V/2A (24W) 给电机/屏幕
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  负载       │ MPU + 屏幕 + 电机 + 麦克风
│             │ 待机: < 0.5W, 工作: 5-15W
└─────────────┘
```

### 3. 功耗预算表 (估算)

| 组件 | 工作电流 @5V | 待机电流 | 备注 |
|------|-------------|---------|------|
| MPU (RK3566/RK3576) | 1.0-2.0A | 50-100mA | 运行 Linux + AI |
| 屏幕 (7" 1024×600) | 0.5-1.0A | - | IPS 液晶 |
| 麦克风阵列 | 0.1A | 10mA | 6 麦克风 |
| WiFi + BT | 0.3A (peak) | 20mA | 蓝牙 + 2.4G WiFi |
| 两个小电机 (移动) | 1.0-2.0A | 0 | 减速电机 |
| LEDs + 传感器 | 0.2A | 5mA | RGB 灯 + IMU |
| **总计** | **3-6A (15-30W)** | **< 200mA (<1W)** | |

**续航计算**:
- 电池容量: 10Ah × 3.2V = 32Wh
- 日均使用 2 小时，功耗 20W → 40Wh → **不足**
- 需 **20Ah 电池 (64Wh)** 或 **降低功耗** (使用更低功耗 MPU)

### 4. "双核" 实现的技术选择

#### 方案 1: Raspberry Pi Zero + Arduino (低成本)
- **MPU**: RPi Zero 2 (1GHz 单核, 512MB RAM) - $15
- **MCU**: Arduino Nano 33 BLE - $15
- **太阳能**: 20W 板 + 10Ah 磷酸铁锂 - $40
- **总成本**: ~$80-100

#### 方案 2: Rockchip RK3566 + STM32 (高性能)
- **MPU**: RK3566 (4核 A55, 2GB RAM) - $35
- **MCU**: STM32F4 (Cortex-M4) - $5
- **太阳能**: 30W 板 + 10Ah 磷酸铁锂 - $50
- **总成本**: ~$100-120

#### 方案 3: Allwinner + ESP32 (极致集成)
- **MPU**: Allwinner V831 (Cortex-A7, 1GB) - $20
- **MCU**: ESP32-S3 (WiFi/BT 内置) - $5
- **太阳能**: 20W 板 + 7Ah 锂电 - $35
- **总成本**: ~$70-80

## 🔋 太阳能供电可行性分析

### 充电能量计算

**假设**:
- 太阳能板: 30W, 效率 85%
- 日照 5 小时/天 (车内透过车窗)
- 电池: 12V 7Ah (84Wh)

**日均发电**: 30W × 5h × 85% = **127.5Wh**

**机器人日均耗电**:
- 待机 22h × 1W = 22Wh
- 工作 2h × 15W = 30Wh
- **总计 52Wh**

**盈余**: 127.5 - 52 = **75.5Wh** (足够且能充放电池)

✅ **结论**: 太阳能供电完全可行，即使阴天也能维持待机
