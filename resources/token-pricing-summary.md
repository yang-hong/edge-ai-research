# 如何买到便宜的大模型 Token - 视频总结

## 📊 Token 消耗真相

| 使用场景 | 每次调用 | 月消耗量 | 月成本 |
|---------|----------|----------|--------|
| 普通聊天（豆包/Gemini网页） | 几千~几万 | ~300万 | ~¥几块 |
| Agent（Claude Code/Open Code） | 几十万~上百万 | ~3亿 | **¥1500+** |

> 💡 **核心矛盾**：Agent 需要持续规划、执行、反思，token 消耗是普通聊天的 **100倍以上**

---

## 🎯 省钱三大取舍框架

**价格 × 稳定性 × 模型能力**
- 不可能三者兼得
- 最强模型+稳定 → 稍贵
- 要便宜 → 牺牲能力或稳定性

---

## 🛒 各平台方案实测

### 1. Claude 4.5 - 性价比之王

**官方订阅**：$100 → 额度 $1350（不到1折）
- 相当于 **¥0.5 = $1** 使用额度
- **坑**：
  - 国内用户易封号
  - 不能接入第三方工具（只能 Claude Code 内使用）

**中转站**（零售商批量买套餐再加价转卖）
- 成本价 ¥0.5/$1，零售价 ¥1+/$
- 市面上上百家，建议少充多试
- 参考稳定性监测网站

**评价**：性能最强，性价比最高，适合不怕折腾的用户

---

### 2. OpenAI (ChatGPT) - 最开放

- 额度比例：**¥0.3 = $1**（比 Claude 便宜）
- **Team 套餐首月免费**：可拼车，¥几块/月得 $25 额度
- Codex 速度稍慢，可多 agent 并行跑
- ✅ 可接入 Claude Code、Open Cloud 等第三方工具
- ✅ 封号风险低，政策更友好

**评价**：开放度最高，适合需要集成的用户

---

### 3. Gemini - 近乎免费（但不稳）

**骚操作**：
1. 教育优惠 → 免费一年 Google AI 会员
2. 用 AntiGravity IDE 跑 Claude 模型
3. 通过 **Clipriproxy API** 转接给 Claude Code/Open Cloud 使用

- 因为 Google 会员容易白嫖，token≈免费
- ❌ 用的人太多，Google 频繁调整 AntiGravity 额度，**极不稳定**

---

### 4. 国产模型 - 真香价格

| 模型 | 价格 | 效果 | 套餐额度 | 缺点 |
|------|------|------|----------|------|
| **Kimi K2.5** | 首月 $0.99（原价 $99） | ≈ Claude Sonnet 4.5 | 较抠门 | 看不到具体用量 |
| **GLM4.7** | ¥20/月（¥240/年） | 稍弱于 Kimi | 很足 | 高峰期慢 |
| **MiniMax** | 类似 GLM | - | - | - |
| **NVIDIA（免费）** | 完全免费 | 提供上述开源模型 | 不限 | 速度巨慢 |

**国产总结**：
- 追求效果 → **Kimi K2.5**（海外版）
- 追求性价比 → **GLM4.7**（最实惠）
- 零成本试试 → **NVIDIA**（但要有心理准备）

---

## 🏆 个人推荐方案

**1. 追求最强效果**
→ Claude 中转站（长期稳定使用）

**2. 预算有限**
→ 国产 GLM 套餐（¥20/月，够用）

**3. 零成本启动**
→ OpenAI Team 拼车（几块钱/月）
→ 或 NVIDIA 免费模型（速度慢但能用）

**4. 最佳组合**
CLAUDE 主跑 + OpenAI 备用 + GLM 日常

---

## ⚠️ 注意事项

- 中转站属于灰色地带，别一次充太多
- Claude 官方封号严重，建议用中转站或海外 IP
- 模型换代快，关注新版本（Kimi、国产都在快速迭代）

---

**省钱的本质是取舍**：根据你的**需求场景**（效果、集成、成本）选择合适的方案组合，没必要死磕一个。
